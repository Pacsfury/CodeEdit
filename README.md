# CodeEdit
A basic text and code editor

---

## What includes
CodeEdit includes basic functionalities for writing code:
* Basic open/save/save as system
* Horitzontal bar for opened files, every file with a close button
* A plugin system

## What are plugins
Plugins are GdScript code that modifies how the editor works. You can find two AI-Generated plugins on the `PLUGINS/` folder or create (and send through a PR) your own plugins.
